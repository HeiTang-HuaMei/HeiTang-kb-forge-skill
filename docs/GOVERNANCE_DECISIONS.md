# Governance Decisions

`governance_decisions.jsonl` records inclusion and exclusion decisions for curated packages.

Each record includes:

- `decision_id`
- `item_id`
- `item_type`
- `decision`
- `reason`
- `reviewer`
- `created_at`
- `source_evidence_refs`
- `output_item_id`

This audit log supports reviewability. It does not claim curated packages are automatically perfect.
