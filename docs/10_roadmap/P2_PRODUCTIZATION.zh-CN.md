# P2 Productization

P2 只能在 P1 UI Core Parity 有证据后开始。它是未来工作，不是当前 v4.0 声明。

## 范围

- packaging 和 installer polish
- release note 纪律
- 用户侧 troubleshooting 和 diagnostics refinement
- 文档发布治理
- 最终产品验收循环
- optional provider UX hardening，且不提交 secrets

## 未证明前不属于范围

- SaaS
- team permission systems
- platform-hosted user data
- cloud sync
- 完整 external vector database production readiness
- v4.0 release 或 tag
