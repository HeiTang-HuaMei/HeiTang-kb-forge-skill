# P1 UI Core Parity

P1 是 v4.0 可以启动前的下一道必要产品门禁。它尚未完成。

## 目标

证明 UI 能引导或操作 Core 已证明的主要 workflow：

- workspace setup
- file selection
- KB build
- query and verification
- document generation
- Agent and Skill creation
- local runtime flows
- storage and memory lifecycle views
- release and gate report review
- provider settings，且不提交 secrets

## 验收边界

P1 需要独立 UI full-operation acceptance gate。Contract view 或 minimal bridge wiring 不足以声明完整 Workbench operation。

## 当前状态

- Core pre-v4 RC readiness 已完成。
- UI full-operation 仍然 blocked。
- 没有真实证据前不得启动 v4.0。
