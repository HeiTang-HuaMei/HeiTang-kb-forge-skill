# 项目概览

HeiTang Knowledge Workbench 是一个本地优先的知识工作台 Core。它把本地资料整理成可追溯、可检索、可验证、可复用的知识资产，并继续支持文档输出、Skill 生成和 Agent 创建包。

当前公开主分支只代表 v4.2 产品基线。旧 RC、旧 Campaign 中间证据、临时审计输出和历史实验记录不再保留在 main；历史可通过 Git history 和 tag 查询。

## 当前定位

- 产品名称：HeiTang Knowledge Workbench
- Python import namespace：`heitang_kb_forge`
- 当前产品基线：v4.2
- 当前 Core package 版本：4.2.0
- 默认运行方式：本地 CLI / Core package
- 默认网络要求：无
- 默认密钥存储：无

## 核心产物

- Knowledge Package
- Document Outputs：Markdown / DOCX / PDF / PPTX
- Skill Outputs：Skill Template / Skill Suite
- Agent Creation Package
