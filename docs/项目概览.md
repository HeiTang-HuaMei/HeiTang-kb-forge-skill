## 产品一句话

HeiTang Knowledge Workbench 是本地优先的知识资产工作台，帮助用户把资料转成文档库，再构建知识库、索引和 RAG 能力，并通过编排层生成文档、Skill、Agent 和 A2A 协作结果。

## 当前主链路

```text
文档库 → 知识库 → 索引层 → RAG → 编排层 → 文档/Skill/Agent/A2A
```

完整产品链路包含工作区、资料导入、标准知识包候选层和治理配置：

```text
工作区
  → 资料导入
  → 文档库
  → 标准知识包 / OKF 候选层
  → 知识库
  → 索引层
  → RAG 与验证层
  → 编排层
  → 文档 / Skill / Agent / A2A 产物
  → 产物中心 / 审计 / 导出
```

## 当前仓库角色

本仓库保留 `heitang_kb_forge` Core package、CLI、测试、打包脚本和产品文档。公开产品名称可以使用 HeiTang Knowledge Workbench；Python import namespace 继续使用 `heitang_kb_forge`，避免破坏现有 CLI、测试和安装路径。

## 核心产物

- Knowledge Package
- Document Outputs：Markdown / DOCX / PDF / PPTX
- Skill Outputs：Skill Template / Skill Suite
- Agent Creation Package
- Agent / A2A 相关内容以 v3 基线定义的编排产物为准，旧 Agent Runtime 词汇只作历史边界说明
