# Export Certification

v2.5 对 OpenClaw、小红书、Codex、Claude Code、MCP、generic、local registry 和 all-platform 本地导出包做发布认证检查。

输出 `platform_export_certification.json`、`platform_export_certification_report.md` 和 `platform_certification_findings.jsonl`。

认证检查必要文件、platform manifest、安装说明、上传说明、policy 文本、mock publish 边界、疑似密钥、危险命令和不支持的 runtime 声明。

它不上传包，也不运行平台 runtime。
