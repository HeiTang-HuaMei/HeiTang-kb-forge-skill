# 外部运行时参考队列

本队列只记录 `reference_only` / `needs_verification` 参考，不代表当前已接入。

## 当前规则

- 外部项目不得写成已接入 runtime。
- 未验证能力不得写入当前 PRD 完成项。
- Provider、插件、Agent runtime、Computer Use 或 shell 能力必须单独走安全和验收 gate。
- OKF 只作为标准知识包候选层，不作为外部 runtime。

## 最小字段

| 字段 | 含义 |
| --- | --- |
| 项目 | 外部项目或标准名称 |
| 状态 | `reference_only` / `needs_verification` / `verified_candidate` |
| 参考点 | 可借鉴的能力或设计 |
| 当前边界 | 为什么不是当前接入 |
| 下一步 | 是否需要核验、设计或拒绝 |

## 当前结论

本轮 docs gate 不新增外部运行时，不授权插件市场，不开放 arbitrary shell，不启用 Computer Use。
