# 登记项目治理

登记项目只作为参考、候选或待核验材料，不得写成当前已接入。

## 状态词

- `reference_only`：只作为方法、架构或交互参考。
- `needs_verification`：需要后续核验，不进入当前 runtime。
- `verified_candidate`：已经独立核验，可进入后续设计或验收候选。
- `rejected`：不进入当前路线。

## 禁止写法

- 已接入。
- runtime ready。
- 当前默认启用。
- 官方集成完成。
- 自动进入 Provider / 插件市场 / 工具链。

## 使用方式

登记项目可以影响产品讨论、测试样例和未来路线，但必须经过单独 PRD、架构、验收和实现 gate 才能成为当前能力。
