# 能力状态

当前项目版本：2.5.0-alpha.1

HeiTang KB Forge 使用明确的能力状态分级，避免 README、Roadmap 和发布说明夸大实际实现状态。

## Stable / 稳定可用

稳定能力已经实现、经过测试，适合本地离线使用。

- 知识包生成
- Contract v2
- Governance / Retrieval / Evidence Gate
- Skill Package Generator
- Agent Package Generator
- Workspace Registry
- Quality Score
- Review Workflow
- Release Quality Gate
- Regression Check
- Release Blockers

## Preview / 预览能力

预览能力已经实现并测试，但仍需要更多真实样本、兼容性检查或发布级加固。

- Platform Distribution
- XHS Skill Package mock
- OpenClaw / Codex / Claude Code export package
- MCP stub package
- Batch Governance
- Package Lineage
- Curated Package
- Update Impact

## Experimental / 实验能力

实验能力可以用于本地验证，但不应表述为稳定生产能力。

- Master Skill Learning
- Derived Skill Generator
- LLM Quality Assist
- Prompt Profile Versioning
- Provider Readiness
- Golden Samples
- Compatibility Matrix

## Roadmap / 计划中

以下能力在路线图中，但尚未作为稳定能力实现。

- v2.6 真实 LLM live smoke 与 Provider 安全治理
- v2.7 runtime compatibility smoke
- v2.8 领域 Skill 工厂
- v2.9 飞书 / 个人知识库 / 移动端 / 安装端 / iOS
- v3.x SaaS / 权限 / 团队协作

## Out of Scope / 当前不做

以下能力默认不支持，不能在文档中声明为已完成。

- 自动发布小红书笔记
- 绕过平台审核
- 保存用户真实 API key
- 默认真实调用外部平台
- 默认联网运行
- SaaS 多租户
- 权限系统
