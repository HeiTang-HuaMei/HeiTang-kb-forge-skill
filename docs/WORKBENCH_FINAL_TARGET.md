# HeiTang Knowledge Workbench Final Target

## Positioning

HeiTang Knowledge Workbench is a local-first multi-knowledge-base, multi-agent, multi-model orchestration workbench.

It is not a single Skill, a single CLI, or a simple RAG demo.

The final system includes:

- multi-KB management
- multi-Agent creation and binding
- multi-Agent workflows
- multi-model routing
- isolated Agent memory
- explicitly shared workflow memory
- knowledge asset production
- grounded KB Q&A
- document generation
- knowledge-bound Agent / Skill generation
- external Skill reverse engineering and fusion
- local Workbench UI

## Relationship to HeiTang KB Forge Core

HeiTang KB Forge Core is the knowledge supply-chain core Skill.

It prepares standardized, auditable, reviewable, retrievable knowledge asset packages.

The Workbench orchestrates multiple Skills, Agents, KBs, models, and memory scopes into a full product workflow.

## Core Principle

Agent memory must be isolated by default.

Shared memory is allowed only when an explicit workflow handoff or collaboration policy authorizes it.
