# 黑糖 Knowledge Workbench 工业级功能评审 / 验收矩阵 v3

日期：2026-06-19
定位：功能评审与 Owner 复验清单
目标：在原验收矩阵基础上，补齐索引层、RAG 层、编排层和登记项目治理验收。

---

## 1. 验收分级

| 等级 | 含义 | 推版要求 |
|---|---|---|
| L0 | 页面存在但不可用 | 不允许推版 |
| L1 | 单点功能可用 | 不足以推版 |
| L2 | 链路可用 | 可进入候选验收 |
| L3 | 工业级可用 | 推版标准 |
| L4 | 可规模化运营 | 后续增强目标 |

本版本推版目标：核心模块达到 L3。

## 17. 工业级推版验收场景 v3

```text
1. 启动 EXE
2. 创建普通工作区 W0
3. 配置模型 Provider
4. 配置本地存储
5. 配置 Redis 并测试连接
6. 配置向量库并测试连接
7. 导入 PDF、DOCX、Markdown、图片、网页链接
8. 验证文档库多文件持久化
9. 创建知识库 K1：单文档
10. 创建知识库 K2：另一文档
11. 创建知识库 K3：多文档组合
12. 验证 K1/K2/K3 均有 index metadata
13. 检索 K1
14. 检索 K1 + K2 + K3
15. 检查检索结果引用 KB、文档、chunk
16. 执行外部事实验证
17. 保存人工纠偏
18. 生成 PRD 文档
19. 重新打开文档历史快照
20. 删除单条生成历史
21. 从 K1 生成 Skill S1
22. 从 K2 + K3 生成 Skill S2
23. 导入外部 Skill S0
24. S0 + K2 生成本地化 Skill S3
25. 创建单 Agent 工作区 WA
26. 创建复杂 Agent A，绑定 K1 + S1 + 模型 + 记忆
27. Agent A 对话并显示引用
28. 创建多 Agent 总工作区 WM
29. 创建子 Agent B，绑定 K2 + S2
30. 创建子 Agent C，绑定 K3 + S3
31. 在 WM 启动 A2A
32. A2A 多轮协作
33. 输出共识、冲突点、行动建议
34. 导出 A2A 报告
35. 查看运行审计
36. 删除部分历史和产物
37. 重启 EXE
38. 验证数据、配置、产物仍一致
```

## 18. 推版阻断项

出现以下任一问题，不应推版：

1. EXE 启动白屏或黑屏。
2. 导入新文件覆盖旧文件。
3. 文档库无法持久化。
4. 文档库没有真实解析正文。
5. 知识库无法从文档库选择来源。
6. 知识库没有 index metadata。
7. 知识库产物无法查看。
8. 检索结果无法追溯引用。
9. RAG 回答没有引用或证据。
10. 文档生成模板不生效。
11. 文档历史无法删除或重新打开。
12. Skill 无法导出或无法绑定 Agent。
13. 外部 Skill 无法解析或无法本地化。
14. Agent 创建后不能对话。
15. Agent 回复不显示 KB / Skill 来源。
16. Agent 越权访问未绑定资源。
17. 多 Agent 没有总工作区。
18. 子 Agent 没有独立工作区。
19. A2A 结果无法保存。
