# 路线图

当前路线图按 v3 三基线的 P0 / P1 / P2 组织，旧 Campaign 计划只作追溯。

## P0

- 工作区、导入、解析、文档库。
- 知识库创建和基础索引。
- RAG 检索、引用和验证报告。
- 文档、Skill、Agent、A2A 产物。
- 产物中心、审计中心和导出。

## P1

- 多 KB 复制、合并、拆分、增量更新和回滚。
- 向量索引、记忆索引和外部向量库配置。
- RAG 查询改写、重排、覆盖率、冲突检测和人工纠偏。
- Skill 编辑、验证、导出、版本管理。
- Agent 权限审计、对话历史、导出一致性。
- A2A 多轮协作与冲突检测。

## P2

- 标准知识包 Bundle。
- OKF Bundle 导出。
- OKF → 知识库构建。
- OKF + Skill 个性化。
- OKF + Agent 通过 KB / Skill 间接继承来源、标签、版本和引用。
- 规模化治理和高级编排。

本轮 docs gate 不新增功能、不改 runtime、不 tag、不 release，不打 stable v4.3.0。
