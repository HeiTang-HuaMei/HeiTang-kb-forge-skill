Date: 2026-06-17

## Status Semantics

| UI state | Visual treatment | Meaning | Allowed action |
| --- | --- | --- | --- |
| `enabled_real` | available state, not yellow | Accepted runtime or accepted local binding exists. | Entry must navigate to a real execution/status page or accepted local status surface. |
| `display_only` | neutral preview state, not yellow | Information is visible but not an executable runtime surface. | Must not be described as accepted execution. |
| `disabled_boundary` | yellow boundary marker | Intentionally disabled, not implemented, or not authorized. | Must stay non-executable with a visible reason. |
| pending / waiting / not connected | yellow boundary marker | Not accepted or waiting on external authorization. | Must not be marked accepted. |
