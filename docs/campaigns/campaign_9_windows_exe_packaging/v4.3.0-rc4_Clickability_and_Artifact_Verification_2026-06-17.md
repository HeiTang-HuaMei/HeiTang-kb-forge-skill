Date: 2026-06-17

## Verification Method

This gate uses:

- real rebuilt Windows EXE launch and no-crash click smoke for runtime-window evidence;
- real EXE screenshots for nonblank window evidence;
- Flutter widget tests with stable keys for exact page and action semantics;
- status-label review to prevent preview-only or disabled paths from being presented as `enabled_real`.

Windows UIAutomation exposes the Flutter shell primarily as `FLUTTERVIEW`, so exact semantic identity for every EXE control is covered by widget tests and remains subject to Owner manual retest in the packaged EXE.
