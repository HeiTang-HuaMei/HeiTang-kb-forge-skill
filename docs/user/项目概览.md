# 项目概览

当前产品事实以 [docs/product](../product/) 下三份 v3 文档为准。本文只作为 Workbench UI 仓库的用户入口。

## 产品一句话

HeiTang Knowledge Workbench 是本地优先的知识资产工作台，帮助用户把资料转成文档库，再构建知识库、索引和 RAG 能力，并通过编排层生成文档、Skill、Agent 和 A2A 协作结果。

## 主链路

```text
文档库 → 知识库 → 索引层 → RAG → 编排层 → 文档/Skill/Agent/A2A
```

Workbench UI 的职责是把这条链路做成可操作、可查看、可复验的桌面体验，而不是新增另一套产品事实。

## 用户能看到的关键区

- 工作区和配置。
- 导入与解析。
- 文档库。
- 知识库与索引。
- 检索 / RAG / 验证。
- 文档生成。
- Skill 工厂。
- Agent / A2A。
- 产物中心和审计中心。
