# 快速开始

本页只保留当前可跑链路，不写未来能力。当前产品事实以 [docs/product](../product/) 下三份 v3 文档为准。

## 运行桌面工作台

在本仓库中进入 Flutter 工作台：

```powershell
cd web\workbench\flutter_app
flutter pub get
flutter run -d windows
```

## 最小验证路径

1. 启动 Workbench。
2. 创建或选择工作区。
3. 导入真实文件或文件夹。
4. 解析并进入文档库。
5. 从文档库构建知识库。
6. 查看索引 / chunks / quality report。
7. 执行检索或 RAG 验证。
8. 生成文档、Skill、Agent 或 A2A 产物。
9. 在产物中心和审计中心查看记录。
