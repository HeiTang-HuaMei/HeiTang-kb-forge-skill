# Export Certification

v2.5 certifies local platform export packages for OpenClaw, XHS, Codex, Claude Code, MCP, generic, local registry, and all-platform export.

It writes `platform_export_certification.json`, `platform_export_certification_report.md`, and `platform_certification_findings.jsonl`.

Certification checks required files, platform manifest, install guide, upload guide, policy text, mock publish boundary, secret risks, dangerous commands, and unsupported runtime claims.

It does not upload packages or run platform runtimes.
