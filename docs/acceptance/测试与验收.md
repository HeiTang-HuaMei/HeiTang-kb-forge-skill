当前验收以 [功能评审 / 验收矩阵 v3](../product/FEATURE_ACCEPTANCE_MATRIX_V3_2026-06-19.md) 为准。

## 验收主线

```text
文档库 → 知识库 → 索引层 → RAG → 编排层 → 文档/Skill/Agent/A2A
```

每个模块至少要验证真实输入、真实产物、可见失败、可追溯审计和重启一致性。

## UI Fast Gate

按影响范围选择最小相关测试；不因 docs gate 改 Core/UI/runtime。

## 文档专项扫描

- overclaim scan：不得把未来能力写成已完成。
- OKF runtime scan：不得出现 OKF 当前 runtime、一级导航或主链路实现声明。
- no-secret scan：不得明文 secret。
- markdown basic scan：本轮新增链接必须可解析。

## 阻断项

- 无 KB 直接回答。
- 回答没有引用。
- 多 KB 结果不标来源。
- 删除后重启恢复。
- Agent 可访问未绑定资源。
- A2A 没有编排记录。
- OKF 被写成当前 runtime ready。
