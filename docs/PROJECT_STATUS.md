# HeiTang KB Forge Skill Project Status

## Status

v1.0.0 stable Agent knowledge supply chain baseline.

## Positioning

HeiTang KB Forge Skill is an Agent knowledge supply chain foundation. It produces traceable, auditable, reusable knowledge asset packages from multi-format source materials.

It is not an Agent Runtime, Tool Runtime, Web UI, online QA service, or SaaS platform.

## Current Capabilities

- Multi-format ingestion: Markdown, TXT, PDF, DOCX, images, CSV, TSV, XLSX.
- Text-based PDF parsing and optional table extraction.
- Scanned PDF and image OCR fallback.
- Best-effort OCR table extraction.
- Standard 7-file knowledge package output.
- Quality report.
- Opt-in LLM structured extraction and LLM quality report.
- Opt-in RAG export.
- Opt-in embedding and vector export standard files.
- Opt-in Agent Template generation.
- Demo, config, and pipeline workflows.
- Opt-in package validation and downstream export.
- Optional live provider validation structure.

## Stable Boundary

- No Agent Runtime.
- No Tool Runtime.
- No Web UI.
- No real vector database service.
- No online QA service.
- No external service dependency for default tests.
