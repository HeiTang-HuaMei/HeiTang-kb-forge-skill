# 标签命名策略

## 当前结论

- `campaign-1-3-baseline`：不得在当前任务中创建。
- `campaign-1-3-baseline-rc.4`：仅用于清理后的 Campaign 1-3 baseline CI / Release Check 验证。
- `v4.2.x` / `v4.3.x`：保留给真实产品版本发布，不用于战役封口候选。

## 历史纠偏

`v3.0.3-integrated-closure`、`v3.0.4-integrated-closure` 只作为 superseded CI validation tags 记录，不是正式 release tag、baseline tag 或产品版本 tag。

失败的 `campaign-1-3-baseline` 预清理 tag 已由用户手动删除，状态为：

- `superseded_failed_pre_cleanup_baseline_tag`
- `not_final_baseline`
- `not_product_release`
- `no_github_release`
- `deleted_from_remote_and_local`
