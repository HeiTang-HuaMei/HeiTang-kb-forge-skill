# 历史版本说明

main 分支不再保留 pre-v4.2 审计堆、旧 RC gate 报告、旧 Campaign 中间证据或临时 current_run/latest 输出。当前 Core package 版本为 4.2.0，公开主分支表达 v4.2 产品基线。

历史证据保留方式：

- Git history
- 历史 tag
- 需要时由对应 CLI / tests 在临时目录重新生成

不把旧文件搬进 `docs/archive` 或 `artifacts/archive`。main 分支只保留简明中文说明。

## 公开 main 保留内容

- v4.2 产品说明
- v4.0.0 / v4.1.1 / v4.2.0 简明版本口径
- 当前运行状态
- 标签命名策略
- Campaign 1-3 简要总结
- 仓库结构规范
- v4.2 清理清单与映射表

## 不再保留在 main

- 根目录旧 JSON gate / fix / external absorption 报告
- `docs/audits/**`
- `artifacts/audits/**`
- 旧 Campaign 中间审计包
- `artifacts/audits/current_run/**`
- `artifacts/audits/latest/**`

push/tag/CI Green 只表示候选基线验证，不等于商业发布完成。

历史版本表述只保留摘要：`v4.0.0` 是历史稳定线，`v4.1.1` 是上一稳定线，`v4.2.0` 是当前 Core package 版本和 public main 产品基线。
