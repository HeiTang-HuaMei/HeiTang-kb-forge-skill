# 仓库结构规范

## 根目录允许

- `README.md`
- `README.zh-CN.md`
- `CHANGELOG.md`
- `LICENSE`
- `pyproject.toml`
- `.gitignore`
- `.gitattributes`
- `.env.example`
- `provider_config.example.yaml`
- `AGENTS.md`
- `SKILL.md`
- `skill.json`

## 工程关键目录保留英文

- `.github/`
- `heitang_kb_forge/`
- `tests/`
- `scripts/`
- `examples/`
- `assets/`
- `desktop/`
- `packaging/`

## docs 公开面

人读文档使用中文文件名。工程运行时、Python package、tests、GitHub workflow 和构建路径不得改成中文。

## 禁止提交

- 根目录旧 JSON 报告，除 `skill.json`
- `artifacts/audits/current_run/**`
- `artifacts/audits/latest/**`
- `_local_dependency_remediation/**`
- `.heitang_cache/**`
- `tmp/**`
- `build/**`
- `dist/**`
- `.venv/**`
- `node_modules/**`
