# Skill Quality

Enhanced Skill templates add release-oriented files:

- `TASKS.md`
- `INPUT_OUTPUT.md`
- `FAILURE_MODES.md`
- `SAFE_REFUSAL.md`
- `EVIDENCE_USAGE.md`
- `OPERATION_GUIDE.md`
- `RELEASE_CHECKLIST.md`
- `skill_validation_result.json`
- `skill_validation_report.md`

The checks are local file-structure checks. They do not prove semantic correctness.
