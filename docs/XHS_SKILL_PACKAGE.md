# XHS Skill Package

v2.4 supports local XHS Skill package preparation only.

Additional XHS outputs:

- `xhs_skill_package/`
- `xhs_skill_manifest.json`
- `xhs_skill_link_manifest.json`
- `platform_policy.md`
- `violation_risk_checklist.md`
- `mock_publish_result.json`

This package does not log in to XHS, does not use a real XHS account, and does not publish notes automatically.
