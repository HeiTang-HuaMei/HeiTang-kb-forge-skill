# Compatibility Matrix

v2.5 writes a compatibility matrix for knowledge packages, Skill packages, Agent packages, workspaces, and platform exports.

The matrix records generated, validated, exportable, certified, offline_safe, mock_only, requires_real_api, official_api, stub_only, runtime_not_executed, release_ready, known_limits, and next_real_validation_version.

LLM live smoke is reserved for v2.6. Runtime compatibility smoke is reserved for v2.7. Feishu / mobile / installer / iOS work is reserved for v2.9.
