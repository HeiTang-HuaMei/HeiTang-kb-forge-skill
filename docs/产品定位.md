# 产品定位

HeiTang Knowledge Workbench 是本地知识供应链工作台，不是云端 SaaS，不是爬虫系统，也不是完整 Agent Runtime。

## 做什么

- 把本地资料转成标准 Knowledge Package。
- 维护 source trace、evidence map、content hash 和质量报告。
- 支持本地检索、问答、验证和 grounded 文档输出。
- 基于知识库生成可治理的 Skill Template / Skill Suite。
- 生成 Agent Creation Package，供后续 runtime 或工作台绑定。

## 不做什么

- 不默认上传用户资料。
- 不默认接入外部平台账号。
- 不绕平台限制。
- 不把 Agent Package 写成 Agent Runtime ready。
- 不把 UI handoff 写成 Campaign 4 UI 完成。
- 不把 Bridge handoff 写成 Campaign 5 Bridge 完成。
- 不把 Redis / Vector DB future target 写成当前 runtime。

## 四类正式产品产物

HeiTang Knowledge Workbench 的产物面分为四类，互不覆盖：

1. Knowledge Package
2. Document Outputs：Markdown / DOCX / PDF / PPTX
3. Skill Outputs：Skill Template / Skill Suite
4. Agent Creation Package

Document Outputs 是正式产品能力，不是审计报告副产物，也不被 Skill Outputs 覆盖。

## 当前底层能力

- Memory Separation / Knowledge Lifecycle
- Evidence Map / Source Trace
- Retrieval / Verification
- Workspace Partition / KB Access Scope
- External Source Memory & Verification
- Document generation
- Skill generation
- Agent package generation
