# Curated Package

v2.3 can build a curated package from review decisions.

Outputs:

- `curated_manifest.json`
- `curated_chunks.jsonl`
- `curated_evidence_map.json`
- `curated_source_inventory.json`
- `governance_decisions.jsonl`
- `decision_audit_report.md`
- `curation_report.md`

Rejected or ignored chunks are not included in `curated_chunks.jsonl`.
