# Roadmap

## v1.0.0

Stable Agent knowledge supply chain foundation.

Completed scope:

- PDF and OCR table best-effort extraction.
- Package validation and readiness reports.
- Hallucination risk fields.
- Downstream export formats.
- Additional Agent Templates.
- Optional live provider validation structure.
- Stable smoke tests and documentation closeout.

## Post-v1 Candidates

- Better PDF table extraction quality.
- Structured table metadata without breaking chunk schema.
- More live provider implementations.
- More downstream format adapters.
- Real vector database import examples.
- Agent platform integration examples.

## Explicit Non-Scope

- Agent Runtime.
- Tool Runtime.
- Web UI.
- SaaS multi-tenant platform.
- Permission system.
- CRM / order / product catalog production integrations.
