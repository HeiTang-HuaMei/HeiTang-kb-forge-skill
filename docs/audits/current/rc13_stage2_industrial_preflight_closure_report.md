## EXE Launch Smoke Hardening

The independent launch smoke gate now rejects placeholder EXE reports. Passing evidence must include:

- `generated_by=scripts/smoke_windows_exe_launch.ps1`
- `exe_path` ending in `heitang_workbench.exe`
- Existing EXE file
- `exe_header=MZ`
- Matching `exe_size_bytes`
- 64-character lowercase hex `exe_sha256`
- `launched=true`
- `crashed=false`
- `startup_timeout=false`
- `secret_plaintext_written=false`

The runtime status check confirms all required launch-smoke predicates passed.
