# 路线图

当前路线图按 [PRD v3](product/PRD_V3_2026-06-19.md) 和 [功能评审 / 验收矩阵 v3](product/FEATURE_ACCEPTANCE_MATRIX_V3_2026-06-19.md) 的 P0 / P1 / P2 组织。旧 Campaign 只作历史追溯，不再作为当前产品路线事实源。

历史当前 v4.2 基线仍作为 Core 兼容口径保留，用于说明 4.2.0 的已完成本地能力，不替代 v3 产品基线。

## P0 当前主链路

P0 聚焦用户能完成真实闭环：

- 工作区创建、切换、删除、持久化和最小配置。
- 文件、文件夹、网页链接、外部 Skill 的导入与解析。
- 文档库持久化、搜索、预览、chunks 查看、关联 KB、删除影响提示。
- 从文档库创建一个或多个知识库，生成 manifest、chunks、quality report 和 index metadata。
- 构建关键词、元数据、引用等基础索引。
- 单 KB / 多 KB 检索、引用证据、验证报告和人工纠偏。
- 文档生成、Skill 生成、单 Agent 创建、A2A 协作产物。
- 产物中心、审计中心、导出和重启一致性。

## P1 增强闭环

P1 强化工业级可用性：

- 多 KB 复制、合并、拆分、增量更新、全量重建、版本比较和回滚。
- 向量索引、记忆索引和外部向量库的配置化启用。
- 查询改写、重排、覆盖率指标、冲突检测、外部事实验证。
- Skill 草稿编辑、验证、导出、版本管理和二次融合。
- Agent 工作区权限审计、KB/Skill 绑定审计、对话历史和导出一致性。
- A2A 多轮协作、冲突检测、审计和报告导出。

## P2 标准化与规模化

P2 作为后续目标：

- 标准知识包 Bundle 导出。
- OKF Bundle 导出。
- OKF → 知识库构建。
- OKF + Skill 个性化。
- OKF + Agent 通过 KB / Skill 间接使用来源、标签、版本和引用。
- 更大规模文档库、多租户治理和高级编排。

## 历史后续战役边界

- Campaign 4：目标导向 UI 工作台。Campaign 4 是产品 UI 工作台，不是 Bridge completion、不是 Agent Runtime completion、不是 EXE packaging。
- Campaign 4 顶层导航不超过七个入口，Agent package spec 不得显示成 runtime ready，Agent runtime 不得显示成 ready，Multi-Agent spec 不得显示成 executable。
- TasteSkill 或 Product Design Plugin 不属于 Campaign 4 base scope。
- Campaign 5：链路级 Local Core Bridge。Campaign 5 是链路级 Local Core Bridge，覆盖 `import_to_kb_flow`、`external_source_verification_flow`、`agent_package_generation_flow`、`agent_output_verification_flow` 等用户任务链路，但不得允许 arbitrary shell execution。
- Campaign 5 可复用 `generate-agent` 与 `validate-agent-package` 检查 Agent Creation Package 产物；`run-agent-task`、`agent-memory-read` 属于 Campaign 6，不得进入 Campaign 5 Bridge allowlist。
- Campaign 6：Agent Runtime / Memory。Campaign 6 才处理 `simple_single_agent_mode_runtime`、`advanced_single_agent_mode_runtime`、Agent memory isolation、Redis / Vector DB memory runtime。Redis config 或 Vector DB config alone is not Agent Runtime & Memory acceptance。
- Campaign 7：配置系统。Campaign 7/8 之前不得把 `check-agent-runtime`、`check-agent-memory-backend`、`check-opencli` 的存在当成验收通过。
- Campaign 8：Full Testing / Review。Fast Gate、focused tests、scoped tests 或单条 green command 不等于 Full Review。
- Campaign 9：Packaging / EXE，并需要 real install or run smoke。
- Final Release 只有 Campaigns 1-9 accepted 后才允许开始。
