## 后续战役边界

- Campaign 4 是产品 UI 工作台，不是 Bridge completion、不是 Agent Runtime completion、不是 EXE packaging。
- Campaign 4 顶层导航不超过七个入口，Agent package spec 不得显示成 runtime ready，Agent runtime 不得显示成 ready，Multi-Agent spec 不得显示成 executable。
- TasteSkill 或 Product Design Plugin 不属于 Campaign 4 base scope。
- Campaign 5 是链路级 Local Core Bridge，覆盖 `import_to_kb_flow`、`external_source_verification_flow`、`agent_package_generation_flow`、`agent_output_verification_flow` 等用户任务链路，但不得允许 arbitrary shell execution。
- `generate-agent`、`validate-agent-package`、`run-agent-task`、`agent-memory-read` 属于 Campaign 6，不得在 Campaign 5 提前写成 ready。
- Campaign 6 才处理 `simple_single_agent_mode_runtime`、`advanced_single_agent_mode_runtime`、Agent memory isolation、Redis / Vector DB memory runtime；Redis config 或 Vector DB config alone is not Agent Runtime & Memory acceptance。
- Campaign 7 才处理 `check-agent-runtime`、`check-agent-memory-backend`、`check-opencli` 等配置系统能力。
- Campaign 8 才处理 Full Testing / Full Review；Fast Gate、focused tests、scoped tests 或单条 green command 不等于 Full Review。
- Campaign 9 才处理 Packaging / EXE，并需要 real install or run smoke。
- Final Release 只有 Campaigns 1-9 accepted 后才允许开始。
