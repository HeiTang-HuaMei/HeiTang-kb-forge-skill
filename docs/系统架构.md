# 系统架构

```text
本地资料
-> 文档解析与标准化
-> Knowledge Package
-> 检索 / 验证 / 文档输出
-> Skill Template / Skill Suite
-> Agent Creation Package
```

## 主要层次

- Core CLI：`python -m heitang_kb_forge.cli`
- 数据合同：JSON / JSONL / Markdown / Office 文档输出
- 证据层：source trace、evidence map、content hash
- 生成层：document outputs、skill outputs、agent creation package
- Workbench handoff：只提供后续 UI / Bridge 的合同输入，不代表 UI 或 Bridge 已完成
- Workspace / memory：本地 workspace registry、KB access scope、memory policy report 和 no-cloud 默认边界；Redis / Vector DB 是 future target，不是当前 runtime dependency。
- UI 表现层：desktop / Workbench 是 presentation layer，Campaign 4 UI 未启动。
