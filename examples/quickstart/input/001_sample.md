# HeiTang KB Forge Quickstart Sample

HeiTang KB Forge Skill is an Agent knowledge supply-chain pre-processing Skill.

It can process multi-format materials into a standard knowledge package with chunks, cards, QA pairs, glossary terms, manifest metadata, quality reports, RAG export, Agent Tool declarations, and MCP readiness files.

The package can be used by a Generic Agent, OpenClaw, Claude Code, or Codex before downstream RAG or Agent workflows.

This sample mentions RAG, Agent Tool, MCP readiness, source traceability, and quality evaluation so the quickstart has searchable content.
