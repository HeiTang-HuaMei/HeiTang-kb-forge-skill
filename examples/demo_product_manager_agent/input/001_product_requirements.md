# 产品需求片段：AI 知识库生成工具

目标用户：产品经理、运营、AI Agent 构建者。

核心问题：
- 用户有大量 Markdown、PDF、DOCX、CSV、XLSX 资料，但无法快速整理成 Agent 可用知识库。
- 手动整理知识卡、问答对、术语表耗时高，且难以追溯来源。
- 构建 RAG Agent 前缺少标准化、可评估、可复用的知识资产包。

核心功能：
1. 多格式资料解析。
2. 批量知识库生产。
3. 同序号多文件合并。
4. 知识卡、问答对、术语抽取。
5. 质量报告。
6. RAG 导出。
7. Agent Template 生成。
8. Demo Report 验收。

成功指标：
- 单个知识包生成成功率。
- chunk / card / QA / glossary 数量。
- citation 覆盖率。
- quality_score。
- demo_report final_status。
