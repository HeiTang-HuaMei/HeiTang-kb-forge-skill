# Demo: Product Manager Agent

This demo shows how HeiTang KB Forge turns product documents, user feedback, and metric definitions into an Agent-ready knowledge package.

## Run

Run:

    .\run_demo.ps1

## What it generates

- standard knowledge package
- quality_report.json
- RAG export files
- Agent Template files
- demo_report.md

## Notes

This demo does not call a real LLM, does not write to a vector database, and does not deploy a real Agent.
