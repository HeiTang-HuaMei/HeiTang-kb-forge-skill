# RAG-Anything UI Impact

The UI may show a `reference_only` status, source/license metadata, the local cross-modal schema, and the benchmark profile on Retrieval Verification and Reports/Audit surfaces.

It must not show a ready RAG-Anything runtime, a multimodal query action, a passed runtime quality score, Campaign 3 acceptance, Campaign 3.0 activation, or Campaign 4 completion.
